# 小棋室配對伺服器

這個伺服器負責「等待配對」：把兩個選到相同棋盤、路數、玩法、棋力感覺的人配成同一個房間。

本機測試：

1. 執行 `start-match-server.bat`。
2. 打開棋盤小棋室。
3. 開兩個瀏覽器頁籤，選一樣的配對條件。
4. 兩邊都按「等待配對」。
5. 第一個頁籤會當黑棋房主，第二個頁籤會當白棋加入。

預設配對伺服器網址是 `http://127.0.0.1:8787`。如果之後把伺服器部署到公開網址，可以在瀏覽器主控台設定：

```js
localStorage.setItem("go-board-room-match-server-url", "https://你的配對伺服器網址")
```

GitHub Pages 只能放網頁，不能常駐這個伺服器；要讓陌生玩家真的互相配對，需要把 `match-server.js` 部署到可以常駐執行 Node.js 的服務。
